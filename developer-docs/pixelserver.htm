PiXeL Server

The PiXeL Server manages PiXeL Spaces, Applications should first prepare videoframes, images, sprites, icons etc. in the PiXeL Space, and then request the VideoCard server to transport your PiXeL Space to the Videocard's memory.

00h - 0Fh Reserved
10h - Request PiXeL Space
11h - Reserved
12h - Get PiXeL Space properties
