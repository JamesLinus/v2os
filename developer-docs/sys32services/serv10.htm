System32 Function 10 (0Ah) - Get FileInfo

Description
Gets information on a filename specified in EDI
Input

AL

 10

DS:EDI

 Pointer to a Filename
Output

AL

 0 if Successfull, >0 is errorcode

EDI

 Preserved (Filename)

DL

 Physical disk on which the file is stored

EBX

 Datastart

ECX
