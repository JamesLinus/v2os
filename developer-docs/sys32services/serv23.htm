System32 Function 23 (17h) - Wait for Keypress

Description
Just an usefull, easy service.
System32 first waits until NO key is pressed, and then until a key IS pressed.
Input

AL

 23d (17h)
Output

None

 None
Errorcodes

None

 None
Notes
This function does not clear the key from the keyboardbuffer, it can be retrieved with service29d
