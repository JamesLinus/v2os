System32 Function 41 (29h) - HW Cursor Flashing

Description
Allows you to enable/disable the flashing of the hardware cursor... (Disabling stops it from corrupting your graphical applications)
Input

AL

 29h

AH

 0 = off, 1 is on
Output

None

 Always successfull
