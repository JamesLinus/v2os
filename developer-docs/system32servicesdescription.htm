System32 Services Description
Ordered by Number

00d - Null Function
01d - Set Interrupt Vector
02d - Reserved
03d - Execute a RealMode Interrupt
04d - Send text to the standard outputdevice
05d - Send hex-number to standard outputdevice
06d - Read physical disk sector(s)
07d - Write physical disk sector(s)
08d - Request Selector
09d - Get partition information
10d - Get FileInformation
11d - Create New File
12d - File part/name Splitter
13d - QBoot
14d - Allocate Memory+selector
15d - Delete Selector
16d - Get Module Info
17d - Announce Server
18d - Remove Server
19d - Query Amount of Servers
20d - Get Server-Module
21d - Call ModuleServices
22d - RM Int (Without Register Copy)
23d - Wait for Keypress
24d - Set V2_OS 'Look'
25d - Shell Command
26d - Delay
27d - Announce Disk Interface
28d - Diskbuffer Dump
29d - Get Key from Keyboard Buffer
30d - Compare String
31d - UCase String
32d - Get Amount of Devices in PCI(Sub)Class
33d - Get Resources used by a PCI Card
34d - Get I/O Resources used by a PCI Card
35d - Get Memory Resources used by a PCI Card
36d - Get IRQ Mask
37d - Set IRQ Mask
38d - Write text to alternate screen
39d - Write number to alternate screen
40d - Read text
41d - Cursor Flashing On/Off
42d - Get Interrupt Vector
43d - Get Selector's base address
44d - Set Selector limit
