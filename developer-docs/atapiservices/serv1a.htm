ATAPI Function 26 (1Ah) - File Info

Description
Gives information on the existence, location and size of the specified filename.
Input

AH

 19h

ESI

 devicecode ('cda',0 or 'cdb',0 etc.)

EBX

 Pointer to a filename (Must be ucase !)
Output

AL

 0 is Ok, > 0 is filenotfound...

EBX

 StartSector

ECX

 Filesize in Bytes
Errorcodes

None

 None
Notes
None.
