VideoCard Server

The VideoCard Server is the interface to your VideoCard Hardware. It has very little to do with graphics, it only does the hardware communication with the card. It provides services like card initialization / detection, VideoMode changing Hardware scrolling, color ramp programming etc.

00h - 0Fh Reserved
10h - Get Compatibility Version
11h - Initialize Card
12h - Check if videomode is supported by hardware
13h - Set VideoMode
14h - Initialize FrameBuffer
15h - FrameBuffer to VideoMemory
