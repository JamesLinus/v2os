ATAPI Server

The ATAPI Server gives access to the CDRom drives, and to files on a ISO9660 disc.

00h - 0Fh Reserved
10h - Ident
11h - Get Media Status
12h - Get Track Info
13h - Play CD-Audio
14h - Eject CDTray
15h - Read TOC
16h - Display TOC
17h - Get Disc Info
18h - Read Sector
19h - Reset ATAPI Devices
1Ah - Get FileInfo
