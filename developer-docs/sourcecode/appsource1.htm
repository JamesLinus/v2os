Application Sourcecode #1
This small application will print the legendary words 'Hello World !!!' to the screen, wait for a keypress and returns to the Command-Prompt.

;---------------------------------------------------
; Hello World !!! for V2_OS
; (c)1999 The V2_Lab
;---------------------------------------------------
.MODEL TINY ; Some compiler directives...
.386p

code32 segment para public use32 ; Declare the 32 Bit Code/DataSegment
code32 ends

code32 segment para public use32 ; Start the 32 Bit Code/DataSegment
assume cs:code32, ds:code32

ORG 100H ; Reserve 100h Bytes

HelloWorld: ; Program Starts here
jmp Start ; Skip the variable declaration part
; Declare your variables here...
Message DB 'Hello World !!!',13 ,10 ,0
; On with the code:
Start:
; Prepare the 'Print' Request
Mov AL, 4H ; System32 Service Number 4 (Print Text)
Mov EDI, Offset Message ; Point DS:EDI to the text
Int 20h ; Execute System32 Service


Mov AL, 23 ; System32 Service Number 32 (Wait for a key)
Int 20H ; Execute System32 Service

RETF ; Return to System32

code32 ends ; End the 32Bit Code/DataSegment

END HelloWorld ; End the Sourcecode
